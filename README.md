# Sergik776 конфигурация Arch Linux

На данный мемент содержит конфигурации для:

hyprland - тайлинговый композитор
hyprlock - блокировщик экрана для hyprland
hyprpaper - утилита для управления обоями для hyprland
waybar - статусная панельдля wayland
mako - менеджер уведомлений
wofi - меню приложений для wayland
yazi - tui файловый менеджер
